git status

It should show this

test.html

file. Now type

git add .

So this adds this file as something you have changed and are ready to be merged into the main application. Now type

git commit -m "SOME MESSAGE ABOUT WHAT YOU'VE CHANGED (ie. Added my name)"

So yeah, change the commit message to something describing what's changed. Once that's all set, type

git push

and it should return something saying a new branch in the remote repo has been made. Awesome!
Now, you go to Github to the this project's repo and make sure you are signed in. So there will be a notification somewhere on the page saying to open a pull request with a green button. Press that, and your commit message should be there and the page should show what changes you've actually made in code. If that's what you want, go all the way to the bottom and press to open the pull request.
Github is awesome with letting other people review code before it gets merged into the main app. Tons of companies use Github to manage their projects. Google, Airbnb, MongoDB, Wordpress, so many. Once it's been approved and merged in, then yay your new code is in the master version!
